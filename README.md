# Battlestag Vault

Personal knowledge base and task management system.

## Structure

- **Tasks/** - Action items and task tracking
- **Projects/** - Project notes and documentation
- **Daily/** - Daily notes and logs

## Automation

This vault is automatically updated by:
- com-observer: Extracts tasks from Slack/Gmail/Trello
- Other agent programs as they are added

Synced via git and Obsidian Sync to all devices.
